# Paper Artifacts

This folder contains the **publication-ready figures and tables** that
appear in the manuscript. All files in this folder are auto-generated by
`../notebooks/figures/ot_ics_ids_paper_figures_v2.ipynb`; do not edit them
by hand.

paper_artifacts/
